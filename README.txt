GitHub Pages:
1. Pakk ut ZIP-filen.
2. Last opp alle filene i repositoryet.
3. Commit changes.
4. Gå til Settings → Pages.
5. Source: Deploy from a branch.
6. Branch: main.
7. Folder: /root.
8. Save.
